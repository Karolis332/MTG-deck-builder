## Frank Karsten Mana Math Quick Guide

**How many colored sources do you need?**

- **Mono-color:** 16–18 sources (very easy)
- **2-color:** 24–26 sources (mixed across colors)
- **3-color:** 28–32 sources (high fixing required)
- **4-color:** 32–36 sources (maximum fixing)
- **5-color:** 36+ sources (heavy ramp + optimal dual lands)

### Mana Source Breakdown
- **Lands:** Typically 24 (Standard), 35–40 (Historic/Commander)
- **Mana Rocks:** 4–6 (colored rocks count toward colored sources)
- **Mana Dorks:** 2–4 (creatures that produce mana)

**Example (Historic Brawl 3-color):**
- 38 lands + 5 Arcane Signets (mixed) + 2 dorks = ~50 colored sources total
- Per color: ~10–12 sources (reasonable for 3-color)

**Tools:** Use [Scrollvault Mana Base Calculator](https://scrollvault.net/tools/manabase/) for exact calculations.

## Key Principles

1. **Lands first:** Get your land count right (24–26 for 60-card, 35–40 for 100-card)
2. **Ramp enables consistency:** 8–12 sources in Historic/Commander, 5–8 in Standard Brawl
3. **Draw prevents flooding/drought:** 8–12 sources in 100-card, 5–8 in 60-card
4. **Removal density:** 8–15 total interaction (spot + wipes) prevents being overrun
5. **Mana curve:** Most decks peak at 2–3 CMC; adjust lands if curve is heavy
6. **Archetype matters:** Control needs +2 lands and +4 removal; Aggro needs −4 lands and +8 creatures

---

## Color Identity Adjustments

- **Mono-color:** Standard baselines (best consistency)
- **2-color:** May run −1 land if mana rocks are strong
- **3-color:** Need stronger dual lands, may add +1 land
- **4-5 color:** Need optimal fixing; add +1–2 lands and color-fixing rocks
